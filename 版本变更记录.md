# 版本变更记录

## v1.2.0 - 2026-06-23

### 新增

- 新增需求待规划池，可将未确认版本的需求后续分配到当前版本。
- 新增需求范围筛选和状态筛选，支持当前版本、待规划池、当前项目全部三种视图。
- 新增资金四级穿透视图，展示项目总预算、年度预算、版本预算、需求预算和实际消耗。
- 新增资金流水关联需求能力。
- 新增流程里程碑页面，按宏观规划、规划细化、建设落地、招投标、交付验收、运维运营展示成果物补充状态。
- 新增跨版本比对能力，展示版本预算差异和需求新增、移除、变更情况。
- 新增成果物上下文过滤和业务阶段展示。
- 新增 `selftest.py`，支持一键运行全页面冒烟测试。
- 新增统一页头和当前上下文摘要，展示项目、年度、版本、基线状态和当前角色。
- 新增首页状态卡片联动、需求列表双击详情、资金穿透图节点双击下钻。

### 优化

- 优化整体桌面端页面风格，增强指标卡、表格、状态底色、按钮和布局层次。
- 优化左侧导航，当前页面高亮显示。
- 优化通用表格，支持点击表头排序。
- 首页工作台新增预算执行、待规划需求、需求状态分布等关键指标。
- 搜索中心扩展为项目、年度、版本、负责人、预算、状态的综合查询结果。
- 表单新增必填校验和数字校验，降低错误数据写入风险。
- 修复 `说明文档.md` 中文编码异常，补充运行和检查说明。
- 系统设置新增变更申请列表，可审批或驳回冻结版本内的需求变更申请。

### 修复

- 修复弹窗未等待用户点击保存就返回的问题，保证新建和登记类操作真正可用。
- 冻结版本后禁止直接新增需求，提示走变更流程。

## v1.1.0 - 2026-06-08

### 新增

- 新增 `ConsultingRequirementManagerSetup.exe` 安装器。
- 双击安装器后可选择安装路径。
- 安装器会把主程序复制到目标目录，并创建 `data`、`attachments`、`backups`、`exports` 等运行数据目录。
- 安装完成后可选择立即启动主程序。
- 源码新增 `installer.py` 和 `ConsultingRequirementManagerSetup.spec`。

### 调整

- `exe` 目录调整为只保留安装器，便于用户直接双击安装。
- `source` 目录保留源码、README、打包配置和 Git 仓库。
- README 增加安装器打包说明。

### 说明

- 主程序仍为 `ConsultingRequirementManager.exe`，由安装器复制到用户选择的安装目录。
- 运行数据默认保存在安装目录下的 `data` 文件夹。

## v1.0.0 - 2026-06-08

### 新增

- 依据需求文档生成 Windows 桌面 MVP。
- 支持本地 SQLite 数据库首次启动初始化。
- 支持项目、年度计划、落地版本、需求任务、资金流水、成果物管理。
- 支持需求状态流转、角色视图、全局搜索、CSV 导出、本地备份恢复和操作日志。
- 生成单文件主程序 `ConsultingRequirementManager.exe`。
